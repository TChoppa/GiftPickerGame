# 🎁 GiftPicker – Smart Secret Santa Generator  

> 🎯 A fun and interactive web application to automate Secret Santa gift exchanges with fair and random participant assignment

## 📌 Overview
**GiftPicker** is a web-based application inspired by the **Secret Santa** concept, designed to make team gifting activities seamless, fair, and engaging.

It eliminates manual pairing and ensures that each participant is assigned a unique gift receiver randomly.

---

## 🎯 Problem Statement
In traditional Secret Santa activities:

- ❌ Manual pairing can lead to bias or mistakes  
- ❌ Time-consuming coordination  
- ❌ Lack of transparency and fairness  
- ❌ Difficult to manage large groups  

---

## 💡 Solution
GiftPicker solves these challenges by:

- ✅ Automatically assigning gift receivers randomly  
- ✅ Ensuring no self-assignment  
- ✅ Maintaining fairness and uniqueness  
- ✅ Simplifying group event management  

---

## ⚙️ Features

- 🎲 Random participant assignment algorithm  
- 👥 Add/manage participants  
- 🔒 Unique pair generation (no duplicates)  
- ⚡ Instant result generation  
- 🎉 Simple and interactive UI  

---

## 🧠 Core Logic
- Each participant is assigned exactly one receiver  
- No participant gets themselves  
- No duplicate assignments  
- Based on randomized mapping logic  

---

## 🧰 Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** ASP.NET MVC / ASP.NET Core  
- **Database:** SQL Server (MDF - Local DB)  
- **Architecture:** MVC Pattern  

---

## 📊 Workflow

1. Add participants  
2. Click “Generate”  
3. System assigns gift receivers randomly  
4. Display results
